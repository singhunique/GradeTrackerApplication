# Student Grade Tracking Application

## Project Overview
This project is a **Student Grade Tracking Application** developed as part of the **B202 Advanced Programming** module at Gisma University of Applied Sciences.

The application is a **Java console-based management system** that allows users to manage student records, add grades, calculate averages, and store data using file handling.

---

## Features
- Add new students  
- Add grades for students  
- View all students with grades and average  
- Edit student name  
- Delete a student record  
- Clear all student data  
- Save and load data from a file  
- Console-based menu system  

---

## Technologies Used
- Java
